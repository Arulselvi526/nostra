# nostra - E-Commerce Website

Nostra is a modern and responsive E-Commerce website developed using HTML, CSS, Tailwind CSS, and JavaScript. The website is designed to provide a smooth and user-friendly online shopping experience with an attractive and responsive interface.

# Features
Responsive Navigation Bar
Home Page with Hero Slider
Brands Showcase Section
Customer Experience Section
New Arrivals Collection
Most Wanted Products Carousel
Limited Offer Section
Newsletter Subscription Form
Collections Page with Search and Filter Functionality
Contact Us Page
Mobile-Friendly Design
Interactive Animations using JavaScript
Pages Included

# Home Page

The homepage displays featured products, new arrivals, most wanted collections, promotional offers, and newsletter subscriptions.

# Collections Page

Users can browse products and use search and filter features to find products easily.

# Contact Us Page

A simple contact form allowing users to send inquiries and feedback.
